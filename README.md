# Nest.js + React.js Monorepo

## Introduction

This is a monorepo for the [Nest.js](https://nestjs.com) and [React.js](https://reactjs.org) projects.

## Getting Started

- Start Dev Server

```bash
$ yarn install
$ yarn start:dev
```

- Build for Production

```bash
$ yarn install
$ yarn build
```

- Start Production

```bash
$ yarn install
$ yarn start:prod
```
